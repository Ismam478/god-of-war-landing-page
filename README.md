God of War Landing Page

A responsive front-end landing page inspired by the God of War universe, built using HTML, CSS, and JavaScript.

Overview

This project was built using an AI-assisted workflow (vibe coding) to speed up development while maintaining full control over structure, styling, and final implementation.
The focus was on layout, responsiveness, and clean front-end fundamentals.

Features

Responsive layout for different screen sizes
Clean and structured HTML
Custom CSS styling
Lightweight JavaScript for interactivity

Tech Stack

HTML5
CSS3
JavaScript (Vanilla)

Notes

This project reflects my approach to modern development workflows where AI is used as a productivity tool, not a replacement for understanding.
